## 1. Pages publiques

Ces pages sont accessibles sans compte ni connexion.

| Page | Contenu |
|------|---------|
| Accueil | Presentation generale du centre, carrousel d'images |
| Equitation Harmonique | Philosophie du centre |
| Projet Pedagogique | Methode d'enseignement |
| Amenagements | Installations et equipements |
| Centre de Loisirs | Activites de loisirs |
| Pensions | Services de pension pour chevaux |
| Tarifs | Grille tarifaire complete |
| Cavalerie | Chevaux et poneys du centre |
| Actualites | Nouvelles du centre |
| Planning | Calendrier des cours et evenements |
| Stages | Liste des stages a venir |
| Nous Contacter | Formulaire de contact |
| Revue de Presse | Articles de presse |
| Galeries | Albums photos |
| Plan d'Acces | Localisation et itineraire |

La navigation se fait via la barre de menu en haut de page. Sur mobile, le menu se replie dans un bouton "hamburger" en haut a droite.

## 2. Creer un compte

1. Cliquer sur **S'inscrire** dans le menu
2. Remplir le formulaire :
   - Nom (obligatoire)
   - Email (obligatoire, doit etre unique)
   - Mot de passe (8 caracteres minimum)
   - Confirmation du mot de passe
3. Valider
4. Un email de verification est envoye : cliquer sur le lien recu pour activer le compte

> Si l'email n'arrive pas, un lien sur la page permet de le renvoyer (limite a 6 envois par minute).

A la creation du compte, aucune formule d'abonnement n'est attribuee. Contacter le centre pour souscrire a une formule (annuelle ou carte).

## 3. Se connecter / se deconnecter

**Se connecter :**

1. Cliquer sur **Se connecter** dans le menu
2. Saisir email et mot de passe
3. Cocher "Se souvenir de moi" si souhaite
4. Valider

**Se deconnecter :** cliquer sur **Deconnexion** dans le menu.

## 4. Gerer son mot de passe

**Mot de passe oublie :**

1. Sur la page de connexion, cliquer sur **Mot de passe oublie ?**
2. Saisir l'email du compte
3. Cliquer sur le lien recu par email
4. Choisir un nouveau mot de passe et confirmer

**Modifier le mot de passe (depuis le profil) :**

1. Aller dans **Profil**
2. Remplir : mot de passe actuel, nouveau mot de passe, confirmation
3. Enregistrer

<div class="page-break"></div>

## 5. Gerer son profil

Accessible via le menu **Profil** apres connexion.

**Modifier ses informations :**
- Modifier le nom et/ou l'email, puis enregistrer
- Si l'email est modifie, un nouvel email de verification est envoye

**Supprimer son compte :**
1. En bas de la page Profil, section "Supprimer le compte"
2. Confirmer avec le mot de passe
3. Valider

> La suppression est definitive : toutes les donnees sont effacees (profil, inscriptions, historique).

## 6. Consulter le planning

Accessible depuis le menu **Planning**, sans connexion.

Le calendrier affiche tous les cours, stages et concours. Les evenements sont identifies par couleur :

| Couleur | Type |
|---------|------|
| Brun dore | Cours reguliers |
| Rouge fonce | Stages |
| Bleu fonce | Concours |

**Sur ordinateur** : vue mensuelle en grille, avec possibilite de basculer en vue semaine.

**Sur mobile** : vue en liste.

**Voir les details d'un evenement** : cliquer dessus dans le calendrier. Un panneau lateral s'ouvre avec le nom, la description, les horaires et les boutons d'inscription (si connecte).

## 7. S'inscrire a un cours

**Conditions requises :**
- Etre connecte
- Avoir un email verifie
- Disposer d'une formule d'abonnement active (attribuee par le centre)

**Procedure :**

1. Aller dans le **Planning**
2. Cliquer sur le cours souhaite
3. Choisir le type d'inscription :
   - **Seance unique** : inscription a une date precise
   - **A l'annee** : inscription a toutes les occurrences du cours (formule annuelle uniquement)
4. Cliquer sur **S'inscrire**

**Regles a connaitre :**

- L'inscription a une seance doit se faire **au moins 48 heures avant** le cours
- Si vous etes deja inscrit a la meme seance, l'inscription est refusee
- En formule carte, chaque inscription deduit **1 credit** ; inscription refusee si solde a zero
- L'inscription en ligne aux **stages** n'est pas possible : contacter le centre directement

## 8. Formules d'abonnement

Deux formules, attribuees par l'administration du centre :

### Formule Annuelle

- Inscription **a l'annee** possible : inscrit a toutes les seances d'un cours
- Inscription **a une seance** possible : sans deduction de credit
- Nombre de cours illimite

### Formule Carte

- Fonctionne par **credits** : chaque inscription consomme 1 credit
- Uniquement des inscriptions a la seance (pas d'inscription a l'annee)
- Inscription refusee si solde a zero
- Rechargement de la carte par le centre

## 9. Mon planning personnel

Accessible via le menu **Mon Planning** apres connexion.

Cette page affiche :
- La liste des cours auxquels vous etes inscrit
- Le statut de chaque inscription (active ou annulee)
- Les dates de vos seances
- Votre formule d'abonnement et votre solde de credits (formule carte)

## 10. Se desinscrire d'un cours

1. Aller dans **Mon Planning**
2. Trouver le cours concerne
3. Cliquer sur le bouton de desinscription
4. Confirmer

> En formule carte, le credit n'est pas rembourse automatiquement.

## 11. Consulter les stages

Accessible depuis le menu **Stages**.

La page affiche les stages a venir (dont la date de fin n'est pas passee), tries par date.

**Fiche d'un stage** (cliquer sur un stage) :
- Titre et description
- Dates de debut et de fin
- Programme journalier : activites et exercices prevus pour chaque journee
- Tarifs du stage (ex : demi-journee, journee complete, stage complet)

> L'inscription aux stages se fait en contactant directement le centre.

<div class="page-break"></div>

## 12. Consulter les tarifs

Accessible depuis le menu **Tarifs**, sans connexion.

Les tarifs sont organises en deux onglets : **Cheval** et **Poney**.

Chaque onglet est divise en sections :

| Section | Contenu |
|---------|---------|
| Enseignement | Tarifs trimestriels et forfait annuel |
| Option Serenite | Rattrapages de cours |
| Cartes de seances | Carte 10 seances, Carte Decouverte 5 seances |
| A la carte | Heure passager, cours particulier |
| Proprietaire (cheval uniquement) | Pension box, adhesion annuelle, carte proprietaire |

Certains tarifs affichent un prix barre (ancien tarif) et une frequence de paiement.

## 13. La cavalerie

### Page Cavalerie

Accessible depuis le menu **Cavalerie**. Presente les chevaux et poneys en activite au centre, avec photo, nom et date de naissance.

### Page Hommage

Accessible depuis la page Cavalerie ou le menu. Presente les animaux decedes, dans le meme format.

## 14. Les galeries photos

Accessible depuis le menu **Galeries**.

- La page liste les albums photos, tries du plus recent au plus ancien
- Une barre de recherche permet de filtrer par titre
- Cliquer sur une galerie pour voir toutes ses photos

## 15. La revue de presse

Accessible depuis le menu **Revue de Presse**.

- Liste les articles de presse mentionnant le centre, du plus recent au plus ancien
- Chaque article affiche : titre, resume, date et un lien vers l'article complet
- Une barre de recherche permet de filtrer par titre

## 16. Roles utilisateurs

| Role | Droits |
|------|--------|
| Utilisateur | Consulter les pages publiques, s'inscrire aux cours, acceder a Mon Planning, gerer son profil |
| Enseignant | Droits utilisateur + creer/modifier/supprimer des evenements, consultation avancee du planning |
| Administrateur | Acces a toutes les fonctionnalites de gestion |

Le role est attribue par un administrateur du centre.
